<pan-json-form>

Stable Since 1.0.0

@since 1.0.0

Related: pan-form, pan-schema-form, pan-validation

Attributes

Attribute Type Description
resource string Resource name for PAN topics (default: 'form')
schema string JSON string defining form structure
url string URL to fetch JSON schema from
data string JSON string for initial form values
layout string Layout mode: 'vertical', 'horizontal', 'grid' (default: 'vertical')
auto-validate boolean Validate on input change (default: false)
show-reset boolean Show reset button (default: true)

PAN Topics

ðŸ“¥ Subscribes

  • {resource}.schema.set { schema } - Set form schema
  • {resource}.data.set { data } - Set form values
  • {resource}.data.get - Request current form data
  • {resource}.validate - Trigger validation
  • {resource}.submit - Trigger form submission
  • {resource}.reset - Reset form to initial state

ðŸ“¤ Publishes

  • {resource}.data { data } - Current form data (response to data.get)
  • {resource}.submit { data, isValid } - Form submitted with data
  • {resource}.change { field, value, data } - Field value changed
  • {resource}.validation { isValid, errors } - Validation result
  • {resource}.ready - Form rendered and ready

Examples

      {
        "name": "message",

        "className": "form-control"
      }
    ]
  }'>

  resource="registration"
  url="/api/forms/registration.json"

  auto-validate>

// Programmatic form control
import { PanClient } from '@larcjs/core';
const pc = new PanClient();

// Set form schema
pc.publish({
  topic: 'contact.schema.set',
  data: {
    schema: {
      fields: [
        { name: 'name', type: 'text', label: 'Full Name', required: true },
        { name: 'email', type: 'email', label: 'Email', required: true },
        { name: 'role', type: 'select', label: 'Role', options: [
          { value: 'dev', label: 'Developer' },
          { value: 'designer', label: 'Designer' }
        ]}
      ]
    }
  }
});

// Listen for form submission
pc.subscribe('contact.submit', (msg) => {
  console.log('Form data:', msg.data.data);
  console.log('Is valid:', msg.data.isValid);
});

// Get current form data
pc.subscribe('contact.data', (msg) => {
  console.log('Current data:', msg.data.data);
});
pc.publish({ topic: 'contact.data.get' });
