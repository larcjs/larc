üöÄ Getting Started

All applications are built with PAN (Page Area Network) - a DOM-native message bus for web applications. They work offline, require no build step, and can be deployed anywhere.

To run locally:

# From project root
python -m http.server 8000
# Visit http://localhost:8000/apps/
