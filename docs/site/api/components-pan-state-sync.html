pan-state-sync - Cross-tab state synchronization component
Synchronizes state across multiple browser tabs using BroadcastChannel.
Provides leader election to prevent sync loops and configurable conflict resolution.

Class: PanStateSync

pan-state-sync - Cross-tab state synchronization component
Synchronizes state across multiple browser tabs using BroadcastChannel.
Provides leader election to prevent sync loops and configurable conflict resolution.

Attributes

Attribute Type Description
channel string Name of the BroadcastChannel (default: 'pan-state-sync')
topics string Comma-separated topic patterns to sync (supports wildcards)
strategy string Conflict resolution: 'last-write-wins' | 'merge' | 'custom'
leader string Leader election: 'auto' | 'always' | 'never'
debug boolean Enable debug logging

Examples

<pan-state-sync
  channel="myapp-sync"
  topics="users.*,todos.*"
  strategy="last-write-wins"
  leader="auto">
</pan-state-sync>
