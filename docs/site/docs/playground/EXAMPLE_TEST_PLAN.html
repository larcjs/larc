Test Criteria

  • ✅ PAN: PAN messages flow (check monitor)

2. GraphQL Example

Use a real public GraphQL API: 
{
  name: 'pan-graphql-connector',
  attributes: {
    'endpoint': 'https://countries.trevorblades.com/',
    'resource': 'countries',
    'query': '{ countries { code name } }'
  }
}

3. Form Examples

Add resource and fields attributes: 
{
  name: 'pan-form',
  attributes: {
    'resource': 'users',
    'fields': 'name,email,role'
  }
}

4. Chart Example

Add chart data: 
{
  name: 'pan-chart',
  attributes: {
    'type': 'bar',
    'data': JSON.stringify({
      labels: ['Q1', 'Q2', 'Q3', 'Q4'],
      datasets: [{
        label: 'Sales',
        data: [12, 19, 3, 5]
      }]
    })
  }
}
