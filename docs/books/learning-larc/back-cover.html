Learning LARC
 

The Web Has Grown Up. It's Time Our Apps Did Too.

Modern browsers are fast, secure, and richly capable application platforms—yet most development stacks still treat them like dumb terminals that need layers of tooling, bundling, and framework magic just to function.

Learning LARC shows another path. This book teaches you to build complex, interactive applications using nothing but open web standards: Web Components, ES modules, events, and message buses. No build systems. No monolithic frameworks. No ceremony.

What You'll Learn

  • Design applications as ecosystems of small, cooperating modules that communicate through a shared message bus
  • Build Web Components that are truly reusable and framework-independent
  • Structure large systems using message patterns instead of global state
  • Leverage native browser capabilities without build tooling or bundlers
  • Create loosely coupled architectures inspired by automotive and industrial control systems
  • Develop applications that are maintainable, testable, and platform-native

Who This Book Is For

This book is for web developers who want to build sophisticated applications without the complexity of modern framework stacks. Whether you're maintaining legacy systems, starting fresh projects, or simply curious about what the web platform can do on its own, Learning LARC will help you rethink how modern web applications can—and should—be built.
